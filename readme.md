# pbPlayer 3.4
=============

pbPlayer is cross-browser/cross-platform audio player build upon [pbjs]([https://github.com/Saartje87/pbjs]).

Currently we support

- IE 7+
- Firefox 3.5+
- Opera 9+
- Google Chrome
- Apple Safari
- IOS browsers (iPad, iPhone and iPod touch)
- Browsers on Android-phones and tablets
